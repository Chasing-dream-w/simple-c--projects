Tetris (Console Version)
A C++ console implementation of the classic Tetris game. Use the keyboard to move, rotate, and drop blocks. Features include collision detection, line-clear scoring, and high-score saving.

Controls:

A / D: Move left/right

S: Drop block faster

W: Rotate block

Q: Quit

R: Restart

SPACE: Pause/Resume

C++17 / STL

Supports file I/O for saving high scores



俄罗斯方块（控制台版）
经典俄罗斯方块游戏的 C++ 控制台实现。使用键盘控制方块移动、旋转和下落，具备碰撞检测、消行计分、最高分持久化等功能。

控制方式：

A / D：左右移动

S：加速下落

W：旋转方块

Q: 退出游戏

R：重新开始

SPACE：暂停/继续

C++17 / STL

支持文件读写记录最高分
