# 控制台贪吃蛇游戏

一个用 C++ 编写的控制台贪吃蛇游戏，支持食物分值系统、最高分记录、暂停/退出功能。

##  游戏特性

- 经典贪吃蛇玩法，蛇身用 `■` 表示，蛇头用 `@` 表示，食物用 `*` 表示
- 食物分为 1~5 分，得分越高食物价值越高
- 自动记录最高分并保存到文件 (`highscore.txt`)
- 支持方向键控制（WASD）
- 游戏暂停/继续（空格键）
- 游戏中途退出确认

##  编译与运行

### 环境要求

- Windows 操作系统（因为使用了 `<windows.h>` 和 `<conio.h>`）
- C++ 编译器（如 MinGW、Visual Studio）

### 编译命令（以 MinGW 为例）

```bash
g++ Game_2.cpp -o snake.exe
运行
bash
snake.exe
 注意：由于使用了 Windows 控制台 API，请在 Windows 命令行或 PowerShell 中运行。
```

### 操作说明

按键	功能

W	向上移动

A	向左移动

S	向下移动

D	向右移动

空格	暂停/继续

Q 退出游戏（会确认）

## 游戏规则

控制蛇头移动，吃掉食物 * 可以增加分数

每吃一个食物，蛇身长度增加 1 节

撞到墙壁或撞到自己的身体则游戏结束

游戏结束后会显示最终得分，并自动更新最高分

## 文件说明
Game_2.cpp：游戏主程序源码

highscore.txt：自动生成的最高分记录文件（与程序在同一目录）

## 已知问题
移动刷新使用 system("cls") 和 Sleep()，在部分终端可能出现闪烁

最高分文件若不存在，程序会自动读取为 0
